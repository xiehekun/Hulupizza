# 页面流程

## 堂食扫码流程

```text
扫描桌码
-> 进入菜单页
-> 选择菜品和规格
-> 加入购物车
-> 进入结算页
-> 确认桌号
-> 选择优惠券
-> 微信支付
-> 查看订单状态
-> 后厨制作
-> 服务员上桌
-> 订单完成
```

## 自取流程

```text
打开小程序
-> 选择自取点餐
-> 选择菜品
-> 选择预计取餐时间
-> 使用优惠券
-> 微信支付
-> 获得取餐码
-> 到店核销
-> 订单完成
```

## 后厨处理流程

```text
收到新订单
-> 查看堂食/自取标识
-> 查看桌号或取餐时间
-> 确认制作
-> 标记制作中
-> 出炉
-> 标记待上桌或待取餐
-> 完成订单
```

## 优惠券流程

```text
进入优惠活动页
-> 领取优惠券
-> 进入菜单下单
-> 结算页选择优惠券
-> 系统校验门槛和有效期
-> 抵扣成功
-> 支付
-> 优惠券变为已使用
```
