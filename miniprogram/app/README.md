# 窑烤披萨点餐小程序 MVP

这是第一版最小可用小程序，用本地数据模拟点餐、领券、结算和订单状态。

## 已实现

- 首页：堂食点餐、自取点餐、优惠入口、招牌推荐
- 菜单：分类、商品、规格、加料、加入购物车
- 购物车：数量调整、删除、结算
- 优惠：领取新客券、满减券
- 结算：堂食桌号、自取时间、优惠券抵扣、备注
- 订单：生成订单、查看订单、模拟出餐完成
- 商品管理：新增商品、编辑商品、上架/下架、售罄、删除
- 库存管理：维护当日库存，支付成功后扣减库存

## 如何打开

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择本文件夹：`pizza-kiln-business/miniprogram/app`。
4. AppID 可先使用测试号或当前配置里的游客 AppID。

## 本地检查

在仓库根目录运行：

```bash
node scripts/check-miniprogram.js
```

检查通过后，再用微信开发者工具导入 `miniprogram/app`。

## 第一版边界

- 微信支付暂时用“模拟支付并下单”代替。
- 菜单、优惠券和订单暂时存在本地缓存。
- 商家后台暂未单独实现，当前用订单页和订单详情模拟后厨处理。
- 商品管理暂时使用本地缓存，后续接服务端后可变成正式后台数据。
- 库存扣减暂时发生在“模拟支付并下单”时；接入真实支付后，应移动到微信支付成功回调里处理。
