# 成本与定价模型

## 单品成本拆解

以一张 10-12 寸披萨为例：

| 成本项 | 示例 | 金额 |
| --- | --- | --- |
| 面团 | 面粉、酵母、盐、水、橄榄油 | 待填 |
| 酱料 | 番茄酱或白酱 | 待填 |
| 奶酪 | 马苏里拉、帕玛森等 | 待填 |
| 肉类 | 辣香肠、火腿、培根等 | 待填 |
| 蔬菜 | 蘑菇、罗勒、洋葱等 | 待填 |
| 包材 | 披萨盒、纸袋、餐巾纸 | 待填 |
| 平台费用 | 外卖佣金或支付手续费 | 待填 |
| 单品总成本 | 以上合计 | 待填 |

## 定价公式

建议先用目标毛利率倒推售价：

```text
建议售价 = 单品变动成本 / (1 - 目标毛利率)
```

示例：

```text
单品变动成本 = 22 元
目标毛利率 = 65%
建议售价 = 22 / (1 - 0.65) = 62.9 元
```

## 盈亏平衡估算

```text
月盈亏平衡销售额 = 月固定成本 / 综合毛利率
日盈亏平衡销售额 = 月盈亏平衡销售额 / 营业天数
日盈亏平衡订单数 = 日盈亏平衡销售额 / 客单价
```

## 建议目标

- 披萨单品毛利率：60%-70%
- 综合毛利率：55%-65%
- 外卖平台产品需单独核算，避免佣金吞掉利润
- 开业前至少测算保守、正常、乐观三个销售场景
