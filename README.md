# 窑烤披萨经营项目

这个项目用于梳理和推进一家窑烤披萨店的经营：从定位、菜单、成本、运营、营销到开业执行。

## 项目目标

- 明确店铺定位、目标客群和产品特色
- 建立菜单、配方、出餐和采购的基本标准
- 估算单品毛利、日常固定成本和盈亏平衡点
- 制定开业前 30 天行动计划
- 形成可复制的日常运营流程

## 推荐使用顺序

1. 阅读 `docs/business-model.md`，确认经营方向。
2. 填写 `finance/cost-model.md`，估算价格和利润。
3. 根据 `operations/daily-ops.md` 制定门店流程。
4. 用 `marketing/launch-plan.md` 做开业预热。
5. 阅读 `miniprogram/prd.md`，规划微信点餐小程序。
6. 按 `templates/opening-checklist.md` 推进开店任务。

## 小程序模块

- `miniprogram/prd.md`：餐饮点餐与优惠活动小程序产品需求文档
- `miniprogram/development-roadmap.md`：开发阶段、技术方案和预算粗估
- `miniprogram/page-flow.md`：堂食、自取、后厨和优惠券流程
- `miniprogram/app`：第一版最小可用微信小程序代码

## 核心假设

- 产品类型：意式或美式窑烤披萨，可搭配沙拉、小食、酒水或软饮。
- 经营形态：小型堂食 + 外带 + 外卖。
- 核心竞争力：现烤体验、稳定面团、差异化口味、快速出餐和高复购。

这些假设可以后续根据城市、商圈、预算和店铺面积调整。
